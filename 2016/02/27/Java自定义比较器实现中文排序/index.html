compareTo 方法

　　compareTo()是两个字符串对象比较大小，返回一个整数值，如果调用字符串对象大，返回正整数，反之，返回负整数。相等则返回0。compareTo()是两个字符串对象按ASCII比较大小（汉字是Unicode），返回一个整数值，如果调用字符串对象大，返回正整数，反之，返回负整数。相等则返回0。
